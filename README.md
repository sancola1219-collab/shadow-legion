# 我的方塊世界2：陰影軍團

奪取敵人的影子，建立你的不死大軍！方塊風格 3D 動作收集遊戲。

**🎮 線上玩：<https://sancola1219-collab.github.io/shadow-legion/>**（手機、電腦都可以）

或者雙擊 `index.html` 離線玩，不用安裝任何東西。

進度會自動記住（每 10 秒自動存檔，存在瀏覽器裡）。**離線也會賺**：關掉遊戲，陰影軍團會以半速幫你收集靈魂（上限 8 小時），下次打開直接領！

## 玩法

1. **聖所**：你的基地。奪來的影子站在召喚平台上，每秒自動產出 🔥 靈魂
2. **閘口**：走近紫色傳送門，選擇要挑戰的世界
3. **戰鬥**：每個世界有三個區域＋魔王競技場——殺滿數量開閘門，一路殺到魔王面前！
4. **奪取（Possess）**：打倒敵人有機率把他的影子收為己用；魔王第一次打倒必定奪取！
5. **軍團**：奪來的影子會跟著你出戰（自動戰鬥），也會在聖所掛機產靈魂
6. **商店**：靈魂拿去升級攻擊、生命、奪取率、軍團人數、靈魂加成
7. **重生**：制霸五個世界後可以重生——歸零換永久加成（靈魂 +100%、攻擊 +25%）

## 五大世界

| 世界 | 魔王 |
|---|---|
| W1 陰影墓園 | 骸骨領主 |
| W2 冰霜墓地 | 冰霜君王 |
| W3 熔岩要塞 | 炎獄魔王 |
| W4 毒霧沼澤 | 毒沼霸主 |
| W5 虛空王座 | 虛空皇帝 |

每個世界 5 種敵人（常見/稀有/史詩/傳說），索引圖鑑共 25 種等你蒐集。

## 特色

- 全 3D 方塊世界（自製 WebGL2 引擎，零外部依賴、純前端）
- 第三人稱視角，金甲勇者一刀一個扇形範圍
- 「自動開啟」讓角色自己走自己打，掛機也能推進度
- 敵人頭上有名牌與血條、稀有度顏色，掉落的靈魂紫焰會自動飛過來
- 天氣系統（晴/多雲/雨/雷暴）、五種主題天空
- 稱號系統：達成里程碑解鎖，挑一個配戴
- 手機也能玩：虛擬搖桿＋觸控按鈕

## 操作

| 電腦 | 手機 |
|---|---|
| WASD 移動、滑鼠視角、空白鍵跳 | 左搖桿移動、右半屏滑動視角 |
| 左鍵攻擊 | ⚔ 按鈕攻擊 |
| B 軍團．P 商店．I 索引．T 稱號 | 畫面兩側按鈕 |
| R 自動戰鬥開關．Q 回聖所 | 底部「自動」「返回」按鈕 |
